## Для кого он

Данный репозиторий создан для выполнения задачи 3.1, курса "Автоматизация тестирования с помощью Selenium и Python" на stepik.

## Использование

Сначала перейдите в нужный каталог с помощью команды `cd`.

Создайте виртуальное окружение.
```bash
python3 -m venv .venv
```
Активируйте виртуальное окружение.
Команда для linux.
```bash
. .venv/bin/activate
```
Затем установите зависимости.
```bash
pip install -r requirements.txt
```
Запустите программу с помощью `python3`
